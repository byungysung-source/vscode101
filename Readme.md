# vscode101

Welcome to the vscode101 project! This repository is designed to help beginners learn the basics of Visual Studio Code (VS Code).

## Getting Started

1. Clone the repository: `git clone https://github.com/byungysung-source/vscode101.git`
2. Open the project in VS Code.
3. Explore the files and start coding!

## Features

- Basic setup instructions
- Sample code snippets
- Tutorials on VS Code extensions
